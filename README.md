# 🌦️ Weather App (Python + Tkinter)

A simple and interactive weather application built using Python and Tkinter that fetches real-time weather data using an API.
A beginner-friendly weather app I built while learning APIs and Tkinter

## 🚀 Features

* Get real-time weather by city name
* Displays temperature, humidity, and description
* Dynamic background based on weather conditions
* Weather-based advice (e.g., carry umbrella ☔)
* Weather icons for better visualization
* Error handling with user-friendly messages
* Enter key support for quick search

## 🛠️ Tech Stack

* Python
* Tkinter (GUI)
* Requests (API handling)

## 📦 Installation

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the app:

   ```
   python main.py
   ```
